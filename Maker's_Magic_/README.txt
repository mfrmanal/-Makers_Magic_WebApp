# Maker's Magic - Handcrafted E-Commerce Platform

## Setup Instructions

### 1. How to Import the Database
1. Open phpMyAdmin (http://localhost/phpmyadmin)
2. Create database named `makers_magic`
3. Click "Import" tab
4. Choose `database.sql` file
5. Click "Go"

### 2. How to Run the Project Using WAMP
1. Install WAMP Server
2. Start WAMP (icon should be GREEN)
3. Copy project folder to `C:\wamp64\www\`
4. Open browser: http://localhost/makers_magic/
5. Register an account
6. Login and test features

## Features
- User Registration & Login
- Session Management
- Contact Form with Database
- Custom Gift Orders
